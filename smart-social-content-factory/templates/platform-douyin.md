# 抖音平台 DNA 模板

## 平台画像
- 用户心智："来刷视频的"，耐心极低
- 核心诉求：爽点、笑点、知识点，3秒内被抓住
- 内容形态：15-60秒短视频脚本

## 语言规则
1. **节奏**：信息密度高，一句一个点
2. **情绪**：极端化表达（"绝了"、"崩溃"、"震惊"）
3. **口语**：完全口语化，允许语法不严谨
4. **互动**：强号召（"评论区告诉我"、"点赞收藏防丢"）
5. **钩子前置**：前3秒必须抛钩子

## 脚本结构模板
```
【0-3秒 钩子】
反常识/强情绪/提问
"千万不要再xxx了"
"我真的后悔没早点xxx"

【3-10秒 痛点共鸣】
描述用户正在经历的困境
"是不是你也遇到过..."

【10-30秒 解决方案】
核心产品/方法亮相
"我发现了一个办法..."

【30-50秒 效果证明】
前后对比/用户证言/数据
"你看这个变化..."

【50-60秒 行动号召】
"点击左下角" / "评论区扣1" / "关注我不迷路"
```
