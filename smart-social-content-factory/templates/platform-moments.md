# 朋友圈平台 DNA 模板

## 平台画像
- 用户心智："来看朋友动态的"
- 核心诉求：真实生活、轻度娱乐、社交货币
- 内容形态：纯文字或图文，50-200字

## 语言规则
1. **生活感**：像记录生活，不像打广告
2. **克制**：不刷屏、不硬推、不@品牌
3. **情绪自然**：不过度煽情，不夸张
4. **留白**：说七分留三分，引发好奇
5. **个人化**：带个人习惯和口吻

## 内容结构模板
```
【生活场景】
一个日常瞬间
"今天出门的时候..."
"刚好路过..."

【自然带出】
轻描淡写提到产品/体验
"顺手试了试..."
"朋友推荐了个..."

【轻描淡写的效果】
不夸张，像聊天
"感觉还不错"
"有点意外"

【引发好奇】
不说全，让人来问
"有人要链接吗"
"改天细说"
```

## 禁用
- 直接复制官方文案
- 带#话题#
- @品牌官方账号
- 二维码/链接（除非是正常分享）
- 九宫格全产品图

## 发布策略
- 配1-2张生活化照片（非精修产品图）
- 发布时间：12:00-13:00 或 20:00-22:00
- 频率：同一产品3天内只发1条
