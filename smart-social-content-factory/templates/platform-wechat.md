# 公众号平台 DNA 模板

## 平台画像
- 用户心智："来深度阅读的"
- 核心诉求：有价值、有洞察、有信息量
- 内容形态：长图文，800-2500字

## 语言规则
1. **深度**：有观点、有分析、有数据支撑
2. **结构化**：小标题清晰，段落逻辑严密
3. **可读性**：长短句交错，每段不超过5行
4. **信任感**：引用来源、专家背书、真实案例
5. **节奏控制**：每300字设置一个"继续阅读钩子"

## 内容结构模板
```
【标题】
引发好奇或承诺价值

【开头】
场景引入/痛点共鸣/反常识观点
200字以内，必须让人想继续读

【背景/问题】
交代上下文，为什么这个话题重要
300-500字

【核心观点】
文章中心论点，用加粗突出
200-300字

【论据/案例】
2-3个案例或数据支撑
每段300-500字

【方法论/建议】
可落地的行动指南
300-500字

【结尾】
金句总结 + 情感升华 + 互动引导
100-200字
```

## 排版规则
- 小标题用##或###层级
- 重点句用**加粗**
- 引用用>区块
- 列表用-或数字

## 标签策略
- 文末放关键词标签
- 引导星标/在看/转发
