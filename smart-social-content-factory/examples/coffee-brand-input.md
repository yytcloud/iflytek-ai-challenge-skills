# 示例输入：精品咖啡品牌

## 用户原始需求

帮我给新开的精品咖啡店写一套推广文案，要在小红书和朋友圈发。

## 提取后的结构化输入

| 信息项 | 内容 |
|---|---|
| 主题/产品 | "慢半拍"精品咖啡店，主推手冲咖啡 |
| 目标平台 | 小红书、朋友圈 |
| 内容类型 | 品牌宣传 + 种草推荐 |
| 风格偏好 | 亲切自然、高级质感 |
| 核心卖点 | 云南单品豆、日式深烘、店主是SCA认证烘焙师、店里有只猫叫"咖啡豆" |
| 目标人群 | 25-35岁，追求生活品质的都市白领 |
| 价格区间 | 手冲38-58元，意式25-35元 |
| 店铺位置 | 老城区巷子里，门头不明显 |
