# 示例输入：国货护肤品

## 用户原始需求

我们是一个新国货护肤品牌，主打"植萃修护"，核心成分是积雪草+神经酰胺。新品是修护精华，适合敏感肌。需要在抖音和小红书做种草，还要写一篇公众号深度稿。希望突出"国货不输大牌"这个点。

## 提取后的结构化输入

| 信息项 | 内容 |
|---|---|
| 主题/产品 | "青禾"植萃修护精华，30ml |
| 目标平台 | 抖音、小红书、公众号 |
| 内容类型 | 种草推荐 + 知识科普 |
| 风格偏好 | 专业权威（小红书/抖音）+ 深度洞察（公众号） |
| 核心卖点 | 积雪草+神经酰胺复配、无香精酒精、第三方敏肌测试通过、价格比同成分大牌低60% |
| 目标人群 | 22-30岁，敏感肌、屏障受损、学生党/初入职场的年轻人 |
| 价格 | ¥129/30ml |
| 竞品对标 | 某国际品牌修护精华（¥340/30ml，同核心成分） |
