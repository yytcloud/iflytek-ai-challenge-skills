# 函文种模板

## 适用场景
- 不相隶属机关之间商洽工作
- 询问和答复问题
- 请求批准和答复审批事项

## 结构规范

```
【版头】
  ××××××                              ← 机关名称（可省略"文件"二字）

【主体】
  关于××××的函
  ××××（主送机关）：

  正文：商洽/询问/请求事项
  "现将×××有关事项函告如下："
  或 "为×××，特向贵单位函询/函请×××。"

  具体内容...

  结语：
  商洽类："特此函商。" / "请予函复。"
  询问类："请予函复。" / "特此函询。"
  答复类："特此函复。" / "此复。"
  请批类："可否，请函复。" / "请予批准。"

  ××××××
  2026年×月×日
```

## 常用用语
- "现就×××事宜致函贵单位。"
- "感谢贵单位对×××工作的大力支持。"
- "特此函告，请予支持配合。"

## 格式要点
- 函的语气应礼貌但不过分谦卑
- 不使用指令性用语
- 复函须引用来函标题和发文字号
