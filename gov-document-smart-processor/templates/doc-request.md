# 请示文种模板

## 适用场景
- 向上级机关请求指示、批准

## 结构规范

```
【版头】
  ××××××
  ×××〔2026〕×号
  签发人：×××                            ← 上行文须标注签发人

【主体】
  关于××××××的请示
  ××××（上级机关）：

  正文第一段：请示缘由
  "根据×××，由于×××，拟×××。"

  正文第二段：请示事项
  "一、××××
   二、××××"

  正文结尾：请求批复
  "妥否，请批示。"
  "以上请示，请予批准。"

  附件：×××（如有）

  ××××××
  2026年×月×日
```

## 常用用语
- 开头："根据《×××》规定，为解决×××问题，现就×××请示如下："
- 中间："具体方案如下：..."
- 结尾："妥否，请批示。" / "以上请示如无不妥，请批转执行。"

## 格式要点
- 一份请示只请示一个问题（一文一事）
- 请示不得同时抄送下级机关
- 结尾必须有明确的期复语
