# 示例输入：通知起草

## 用户原始需求

帮我起草一份通知，要求如下：我们局要组织一次安全生产大检查，各个科室和下属单位都要自查自纠，7月底前提交自查报告到安全科。局长办公会要求还要成立专项检查组，8月份进行抽查。
